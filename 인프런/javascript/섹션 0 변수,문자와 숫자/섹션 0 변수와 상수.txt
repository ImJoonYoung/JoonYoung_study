1. 변수와 상수 - 
 1. 데이터를 쓰고 담는 상자
 2. 변수 : var, let  /  상수 : const
 2-1. let은 재할당 가능하지만 재선언은 불가능하다.
 2-2. const는 재할당, 재선언 불가능하다. const는 선언과 할당이 동시에 이루어져야 한다.
**var은 자주 사용하지 않는다!! var은 재할당, 재선언이 가능하다.  나중에 범위가 커지면 어디서 문제가 발생하는지도 모르기 때문이다
 3. 변수명은 명시적으로 누구나 알아볼 수 있게 작성한다.
 4. 예약어 : new, else, do, if, break, for, case, this, catch, finally 등등 변수 작성할때 쓰지않는다.
